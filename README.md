## JartonWebsite
This is the main website for JartonMC. It houses essential information and directs users to other sites if needed (e.g., the store or the wiki).

**Please use your own branch or a dev branch when making significant changes so they can be reviewed!**

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
